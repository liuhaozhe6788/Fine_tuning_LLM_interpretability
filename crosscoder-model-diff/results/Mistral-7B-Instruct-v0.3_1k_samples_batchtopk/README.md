The analysis results of the BatchTopk crosscoder for mistral-7b-instruct model. The training samples are only a subset of 1024 examples from the training dataset.
- `relative_norm.png` stores the relative decoder norm 
- `cosine_sims.png` stores the cosine similarity of decoder vectors between models.
- `base_feature_vis` stores feature visualisations with low relative norm.
- `ft_feature_vis` stores feature visualisations with high relative norm.
